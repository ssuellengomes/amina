# Amina Tech - Mulheres na Tecnologia

## Descrição
Este projeto é um site institucional para a ONG Amina Tech, que tem como propósito criar caminhos reais para que mais mulheres entrem, permaneçam e cresçam na área de tecnologia. O site apresenta informações institucionais, projetos sociais, oportunidades de voluntariado, doações e um formulário de cadastro para novas integrantes.

## Estrutura do Projeto
A organização dos arquivos segue a seguinte estrutura:
 

 /amina │ ├── index.html # Página inicial com missão, visão, valores e contato ├── projetos.html # Página de projetos sociais, voluntariado e doações ├── cadastro.html # Formulário de cadastro de usuárias │ ├── /css │ └── estilo.css # Folha de estilos CSS para o site │ ├── /js │ └── script.js # Script JavaScript para máscaras nos inputs do formulário │ └── /imagens ├── logo.png # Logotipo da Amina Tech ├── missao.jpg # Imagem ilustrativa da missão ├── projetos.jpg # Imagem ilustrativa dos projetos ├── cadastro.jpg # Imagem ilustrativa do cadastro └── certificado.jpg # Imagem de certificado relacionada aos projetos



## Tecnologias Utilizadas

- **HTML5**: Estrutura semântica das páginas, uso de formulários com validações nativas.

- **CSS3**: Estilos externos aplicados para visual limpo, responsivo e paleta de cores suaves e femininas.

- **JavaScript (Vanilla)**: Implementação de máscaras para campos do formulário (CPF, telefone e CEP) para melhor experiência do usuário.


## Como Executar

1. Clone ou baixe o repositório para sua máquina local.

2. Certifique-se de que a estrutura de pastas e arquivos está conforme o especificado.

3. Abra qualquer um dos arquivos HTML (`index.html`, `projetos.html` ou `cadastro.html`) em seu navegador preferido.

4. Para testar o formulário, navegue até `cadastro.html` e preencha os campos. Note que máscaras visuais estarão aplicadas a CPF, telefone e CEP.
